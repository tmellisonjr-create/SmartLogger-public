# SmartLogger-public
SmartLogger is a full-featured ham radio logging program made to leverage the SmartSDR radio hardware by interfacing directly to a FLEX-6000, FLEX-8000 or Aurora radio.  It is SmatLink an multiFlex aware and integrates with the 4O3A PGXL.

A complete overview of everything SmartLogger does — from rapid manual QSO entry to automatic WSJT-X logging, FlexRadio slice tracking, SmartLink support and live radio telemetry, LoTW and eQSL.cc upload and sync, logbook ADIF import, paper QSL card tracking, DXCC entity tracking, space weather, the interactive Grid Map of worked squares, and the Grayline / Spots Globe with real-time 3-D day/night shading and live DX spot visualization.
